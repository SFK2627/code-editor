# Source Modules

This folder is the maintainable source layer for the Sir JR Coding App.

- `src/js/` contains the ordered JavaScript source modules.
- `styles/` contains the ordered CSS source modules.
- `script.js` and `style.css` remain the browser-ready generated outputs.

Edit the source modules first, then run:

```bash
npm run build
npm run check
```

The app still deploys as simple static files, so GitHub Pages and Firebase Hosting stay free-only and easy to upload.
