UPLOAD / REPLACE THESE FILES IN THE WEBSITE REPOSITORY ROOT:

1. index.html
2. style.css
3. script.js
4. firebase-config.js
5. service-worker.js
6. manifest.webmanifest
7. icons/ folder

FIREBASE REQUIRED STEPS:

1. Enable Authentication > Email/Password.
2. Confirm the teacher account exists.
3. Copy firestore.rules into Firestore Database > Rules and publish.
4. Keep the Firebase project on Spark and do not connect billing.

After deployment, hard refresh the website. On an installed phone app, close and reopen it. Clear the old site cache once if the previous layout still appears.
