SIR JR CODING APP — STABILITY + PROFESSIONAL POLISH RELEASE

UPLOAD / REPLACE IN THE WEBSITE REPOSITORY ROOT

1. index.html
2. style.css
3. script.js
4. firebase-config.js
5. service-worker.js
6. manifest.webmanifest
7. favicon.png
8. STUDENT_IMPORT_TEMPLATE.csv
9. icons/ folder, including all four icon files

FIREBASE FILES

10. firestore.rules — publish in Firestore Database > Rules
11. firebase.json — use when deploying with Firebase Hosting

REQUIRED FIREBASE STEPS

1. Enable Authentication > Email/Password.
2. Confirm the teacher account exists.
3. Publish the included firestore.rules.
4. Keep the project on Spark and do not connect billing.

AFTER DEPLOYMENT

1. Hard refresh the website.
2. Close and reopen installed phone PWAs.
3. Clear old site cache once only when the previous version still appears.
4. Test one teacher account, one student account, project save, and live collaboration before the class session.
