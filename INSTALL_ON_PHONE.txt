INSTALL ON PHONE

Android / Chrome:
1. Open the website.
2. Tap the Install button in the app header, or tap the browser menu.
3. Choose Install app or Add to Home screen.

iPhone / iPad:
1. Open the website in Safari.
2. Tap Share.
3. Choose Add to Home Screen.

Important:
Upload these files to GitHub root: index.html, style.css, script.js, firebase-config.js, manifest.webmanifest, service-worker.js, and the icons folder.
