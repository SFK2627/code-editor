INSTALL ON PHONE

Android / Chrome
1. Open the deployed website.
2. Tap the browser menu.
3. Choose Install app or Add to Home screen.

IPhone / iPad
1. Open the deployed website in Safari.
2. Tap Share.
3. Choose Add to Home Screen.

After uploading a new version, fully close and reopen the installed app. Clear the site's stored cache only when the old version still appears after reopening.
